# HTML与CSS注意事项

# html多个文件上传

<input type="file" name="img" multiple="multiple" />

# html accept 属性

在文件上传中使用 accept 属性，本例中的输入字段可以接受 GIF 和 JPEG 两种图像：

<input type="file" name="img" accept="image/gif, image/jpeg" />
